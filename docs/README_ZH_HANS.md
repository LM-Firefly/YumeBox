**一个基于 [mihomo](https://github.com/MetaCubeX/mihomo) 内核的开源 Android 客户端**

</div>


## 特性

- Mihomo 内核
- Sub-Store 可选支持
- Web 面板
- 更多请自行探索...

## 适配性

- Android 7.0 及以上
- 支持 `armeabi-v7a`、`arm64-v8a`、`x86`、`x86_64` 架构

## 使用方法

- **安装**：前往 [安装](https://yumebox.mintlify.app/guide/install)
- **构建**：[跳转至构建章节](https://yumebox.mintlify.app/guide/building)
