## Features

- Mihomo kernel
- Sub-Store Optional Support
- Web dashboard
- For more, please explore on your own...

## Compatibility

- Android 7.0 and above
- Supports `armeabi-v7a`, `arm64-v8a`, `x86`, and `x86_64` architectures
