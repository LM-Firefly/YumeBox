为保持界面简洁，App 内的设置项大多只显示标题，详细说明、类型与默认值集中放在本页。各小节附带「在 App 中打开」按钮，点击即可通过 `yumebox://` 深度链接直接跳转到对应界面。

<Note>
「在 App 中打开」依赖自定义协议 `yumebox://`，仅在已安装 YumeBox 的设备上有效。若点击无反应，可复制链接到浏览器地址栏打开，或在 App 内手动定位。
</Note>

## 首页手势

默认的 Moe 首页支持两种手势，无需进入设置即可操作壁纸与侧栏：

<CardGroup cols={2}>
  <Card title="单击主图" icon="image">
    单击首页顶部的壁纸图片，会直接打开系统相册选择器，选图后进入裁剪界面调整显示范围，确认即更换首页壁纸。
  </Card>
  <Card title="双击主图" icon="panel-left">
    双击壁纸图片，展开或收起左侧信息侧栏（运行时长 / 时钟、快捷入口等）。
  </Card>
</CardGroup>

因此原先「首页侧栏默认展开」开关与「首页壁纸」设置项已从设置中移除——这两项行为现在完全由首页手势接管。

<Card title="在 App 中打开首页" icon="house" href="yumebox://page/home">
  `yumebox://page/home`
</Card>

## 应用设置详解

<Card title="在 App 中打开应用设置" icon="settings" href="yumebox://screen/appsettings">
  `yumebox://screen/appsettings`
</Card>

### 行为

<ParamField path="自动启动" type="boolean" default="关闭">
  应用启动和开机时自动启动代理服务。
</ParamField>

<ParamField path="启动更新配置" type="boolean" default="开启">
  启动时自动更新当前订阅配置。
</ParamField>

<ParamField path="坚持一个中国原则" type="boolean" default="开启">
  自动将台湾地区旗帜及区域码显示为中国。仅在简体中文环境下显示，且固定开启。
</ParamField>

### 主题配色

<ParamField path="主题模式" type="enum" default="跟随系统">
  应用的主题样式：跟随系统 / 浅色 / 深色。
</ParamField>

<ParamField path="反转主题前景色" type="boolean" default="关闭">
  当自动推导出的前景文字 / 图标颜色看起来怪异时，开启此项反转前景明暗。
</ParamField>

<ParamField path="主题配色" type="color" default="#138A74">
  选择主题种子色，App 据此自动推导浅色 / 深色两套配色。
</ParamField>

### 界面

<ParamField path="应用语言" type="enum" default="跟随系统">
  应用语言：跟随系统 / 简体中文 / English。
</ParamField>

<ParamField path="滑动隐藏底栏" type="boolean" default="开启">
  内容向下滑动时自动隐藏底部浮动导航栏，向上滑动时重新显示。
</ParamField>

<ParamField path="背景模糊" type="boolean" default="关闭">
  为顶栏 / 底栏应用模糊背景效果。
</ParamField>

<ParamField path="页面缩放" type="number" default="100%">
  调整 App 整体界面缩放比例，范围 80% – 120%。
</ParamField>

### 首页

<ParamField path="经典首页" type="boolean" default="关闭">
  回退到旧版（经典）首页布局，关闭则使用默认 Moe 首页。
</ParamField>

<ParamField path="首页签名" type="string" default="时间一分一秒流逝而去 终结一步一步迎面而来">
  Moe 首页显示的签名文本。
</ParamField>

<ParamField path="首页签名作者" type="string" default="恋文">
  签名落款。
</ParamField>

### 隐私

<ParamField path="隐藏应用图标" type="boolean" default="关闭">
  隐藏桌面图标后，可通过拨号盘输入 `*#*#0721#*#*` 重新打开 App。
</ParamField>

<ParamField path="隐藏后台卡片" type="boolean" default="关闭">
  启用后，App 不会出现在系统「最近任务」列表（后台卡片）中。
</ParamField>

### 服务

<ParamField path="显示流量通知" type="boolean" default="开启">
  在通知栏中显示实时流量使用情况。
</ParamField>

<ParamField path="单节点测试" type="boolean" default="开启">
  点击节点卡片右侧图标，测试单个节点的延迟。
</ParamField>

<ParamField path="后台隐藏时释放界面" type="boolean" default="关闭">
  当 App 退到后台且界面完全不可见时，主动释放界面以节省内存，同时保留正在运行的代理服务。
</ParamField>

<ParamField path="电池优化无限制" type="系统设置">
  显示当前后台是否受系统电池优化限制；点击跳转系统设置，建议设为「无限制」以降低后台被杀概率。
</ParamField>

### 网络

<ParamField path="自定义 User-Agent" type="string" default="（空）">
  拉取订阅时使用的 User-Agent，留空则使用默认值。
</ParamField>

## 网络设置详解

<Card title="在 App 中打开网络设置" icon="network" href="yumebox://screen/network">
  `yumebox://screen/network`
</Card>

### 代理模式

<ParamField path="路由系统流量" type="enum" default="VPN 模式">
  接管系统流量的代理模式：HTTP 系统代理 / VPN 模式 / Root TUN。
</ParamField>

### 服务配置（VPN 模式）

<ParamField path="绕过私有网络" type="boolean" default="开启">
  绕过私有网络和本地地址，不经代理。
</ParamField>

<ParamField path="DNS 劫持" type="boolean" default="开启">
  将所有 DNS 请求重定向到 YumeBox 处理。
</ParamField>

<ParamField path="允许应用绕过" type="boolean" default="开启">
  允许其他应用通过自身设置绕过本 VPN。
</ParamField>

<ParamField path="运行 IPv6" type="boolean" default="关闭">
  允许通过 VPN 路由 IPv6 流量。
</ParamField>

<ParamField path="VPN 系统代理" type="boolean" default="开启">
  仅在 VPN 模式下，为未走 TUN 的应用设置 HTTP 代理。
</ParamField>

### 服务配置（Root TUN）

<ParamField path="接口名称" type="string" default="Yume">
  Root TUN 创建的虚拟网卡名。
</ParamField>

<ParamField path="MTU" type="number" default="1500">
  Root TUN 链路的最大传输单元。
</ParamField>

<ParamField path="自动路由" type="boolean" default="开启">
  自动添加转发所需的路由。
</ParamField>

<ParamField path="严格路由" type="boolean" default="开启">
  仅允许命中的流量走 Root TUN。
</ParamField>

<ParamField path="自动重定向" type="boolean" default="开启">
  自动启用 Root TUN 所需的重定向规则。
</ParamField>

<ParamField path="DNS 模式" type="enum" default="RedirHost">
  Root TUN 的 DNS 处理方式：RedirHost / FakeIP。
</ParamField>

<ParamField path="FakeIP 地址段" type="string" default="198.18.0.1/16">
  仅在 FakeIP 模式下生效。
</ParamField>

## 深度链接 yumebox://

可在网页、快捷方式或其他 App 中使用以下链接打开 YumeBox 并定位到指定界面。

### 主页面

| 链接 | 目标 |
| --- | --- |
| `yumebox://page/home` | 首页 |
| `yumebox://page/proxy` | 代理 |
| `yumebox://page/profiles` | 配置 |
| `yumebox://page/settings` | 设置 |

### 二级界面

| 链接 | 目标 |
| --- | --- |
| `yumebox://screen/appsettings` | 应用设置 |
| `yumebox://screen/network` | 网络设置 |
| `yumebox://screen/access` | 访问控制 |
| `yumebox://screen/override` | 覆写 |
| `yumebox://screen/providers` | 提供者 |
| `yumebox://screen/connection` | 连接 |
| `yumebox://screen/traffic` | 流量统计 |
| `yumebox://screen/log` | 日志 |
| `yumebox://screen/about` | 关于 |

<Note>
未识别的 host / 路径会被忽略，App 正常停留在当前界面。
</Note>
